# Student Information Card

This program shows a students information when ran correctly. The program uses a plethora of strings, ints, and other commands in java.
## Features

- Displays student name (original)
- Displays major (original)
- Displays graduation year (original)
- Displays email address (New)
- Displays hometown (New)
- Displays hobby (New)
- Added comments that give a description of the program (New)
- Border formatting to make a more clean look (New)
## Technologies Used

- Java 11
- Git for version control
- GitHub for repository hosting

## Installation and Setup

### Prerequisites
Java 11+
Github Desktop app
### Steps to Run

1. Clone this repository
2. Go to the parent directory
3. Use Git bash or BlueJ to compile program the program
4. Use Git bash or BlueJ to run to the program 

## Usage

The program will run and display a mockup of a student info card.
The output shows the original code with the alterations added in the inital code.

### Sample Output:
```
==========================================

==========================================
Name: John Doe
Email: james_emmanuel@students.columbusstate.edu
```
